# Análise de Score de Crédito com Machine Learning

Projeto desenvolvido em Python com foco em Machine Learning para previsão de score de crédito de clientes.

O objetivo do projeto é analisar informações financeiras e comportamentais para prever a classificação de crédito de novos clientes utilizando modelos de aprendizado de máquina.

---

## Tecnologias utilizadas

- Python
- Pandas
- Scikit-Learn
- Jupyter Notebook

---

## Dataset

O projeto utiliza bases de dados disponibilizadas para fins educacionais, armazenadas na pasta:

```text
data/clientes.csv
data/novos_clientes.csv
```

As bases contêm informações relacionadas ao perfil financeiro e comportamento de clientes.

---

## Principais atividades realizadas

- Tratamento e preparação de dados
- Conversão de variáveis categóricas
- Separação entre treino e teste
- Treinamento de modelos de Machine Learning
- Comparação de desempenho entre modelos
- Previsão de score de crédito para novos clientes

---

## Modelos utilizados

- Árvore de Decisão
- Random Forest

---

## Objetivo do projeto

Desenvolver conhecimentos práticos em Machine Learning utilizando Python, aplicando técnicas de preparação de dados, treinamento de modelos e geração de previsões.

---

## Aprendizados

Durante o desenvolvimento deste projeto, foram praticados conceitos relacionados a:
